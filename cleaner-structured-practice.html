Coding Goals

Hope and Vision

Html is a pretty straight-forward language; it is in English and explicit in its intention. The most confusing elements are the semantic ones; features that visually appear the same on the browser (or are not displayed at all, like structural elements) that have important intention and distinctive purpose. My hope is to concretely grasp these topics and be able to code neat, organized, purposeful code.

Further, as an artist (and kind of aspiring UX designer), aesthetic is important to me. I hope to be able to design beautiful interfaces, one's with color, images, shapes, various fonts, columns, and more. I especially want to be able to create dynamic pages, or product motion design, as Payton Meyer advised. It not only interests me artistically, but would be of great value when seeking to get hired.

All in all, I want to create. Art is creation, but above all, communication. Through learning how to communicate via html, I will be able to evolve into a strong, effective creator.
